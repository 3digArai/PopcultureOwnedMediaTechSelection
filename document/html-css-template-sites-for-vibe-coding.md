# Vibeコーディング向け HTML/CSS テンプレートサイト一覧

## 目的
- デザインモックを素早く作るために、HTML/CSSベースのテンプレート入手先を整理する。
- そのまま利用するのではなく、要件に合わせて改修して使う前提。

## 使い分けの目安
- とにかく早くモックを作りたい: `HTML5 UP` / `Start Bootstrap`
- 記事・メディア系を探したい: `Colorlib` / `BootstrapMade`
- 有料でも選択肢を増やしたい: `ThemeForest`

## テンプレートサイト一覧

### 1. HTML5 UP
- URL: https://html5up.net/
- 特徴:
  - 無料で軽量な静的テンプレートが多い
  - 構造がシンプルで改修しやすい
- ライセンス目安:
  - 商用利用を含めて使いやすいが、配布条件は各テンプレートの記載を確認
- おすすめ候補:
  - `Massively`（記事導線を作りやすい）
  - `Editorial`（2カラム寄りでメディア構成に向く）
  - `Phantom`（シンプルで改修しやすい）
- 向いている用途:
  - 記事紹介サイトの初期モック
  - ヒーロー + 記事一覧のベース作成

### 2. Start Bootstrap
- URL: https://startbootstrap.com/themes
- 特徴:
  - Bootstrapベースで実装しやすい
  - レスポンシブ対応が揃っている
- ライセンス目安:
  - 商用利用しやすいテンプレートが多いが、無料/有料で条件差があるため要確認
- おすすめ候補:
  - `Blog/Magazine系のレイアウトカテゴリ`
  - `Landing + Blog構成のテーマ`
  - `シンプルな1カラム記事レイアウト系`
- 向いている用途:
  - 開発速度重視の社内検証
  - 早い段階のUI合意形成

### 3. BootstrapMade
- URL: https://bootstrapmade.com/
- 特徴:
  - ブログ/マガジン系レイアウトが探しやすい
  - 画面パーツが揃っていて拡張しやすい
- ライセンス目安:
  - 商用利用は可能なケースが多いが、クレジット表記有無をテンプレートごとに確認
- おすすめ候補:
  - `Story`（ブログ/記事導線に向く）
  - `Blogy`（マガジン寄りの構成）
  - `Strategy`（特集 + 記事導線を作りやすい）
- 向いている用途:
  - 記事導線のあるブランドサイト
  - サイドバー付きのメディアトップ

### 4. Colorlib（HTML Templates）
- URL: https://colorlib.com/wp/templates/
- 特徴:
  - ジャンル別テンプレートが多い
  - マガジン/ニュース系の例が豊富
- ライセンス目安:
  - 商用利用しやすいテンプレートが多いが、配布形態により利用条件が異なるため要確認
- おすすめ候補:
  - `Magazine`（記事紹介メディア向け）
  - `News/Magazineカテゴリのテンプレート群`
  - `Blogカテゴリのシンプルレイアウト`
- 向いている用途:
  - ポップカルチャー記事紹介サイト
  - カテゴリ導線のあるページ設計

### 5. Cruip
- URL: https://cruip.com/templates/
- 特徴:
  - モダンなデザインが多い
  - Tailwind系テンプレートも選べる
- ライセンス目安:
  - 商用利用可能なものが多いが、無料配布と有料配布で条件が異なるため要確認
- おすすめ候補:
  - `Blogカテゴリのテンプレート`
  - `Magazine/Media向けレイアウト`
  - `Landing + Article導線の組み合わせ`
- 向いている用途:
  - ブランド感を重視したランディング
  - シンプルなトップ + 記事導線

### 6. ThemeForest（HTML）
- URL: https://themeforest.net/category/site-templates
- 特徴:
  - 有料テンプレート数が非常に多い
  - デザイン選択肢が広い
- ライセンス目安:
  - 基本は商用利用前提だが、Envatoライセンス区分（Regular/Extended）を確認
- おすすめ候補:
  - `News/Magazine` カテゴリ上位の高評価テンプレート
  - `Blog + Editorial` 構成の高評価テンプレート
  - `Bootstrap/Tailwind` ベースで改修しやすいテンプレート
- 向いている用途:
  - 要件に近い見た目を短時間で探したい場合
  - 予算を使って初速を上げたい場合

## ライセンス確認チェック（重要）
- 商用利用可否
- クレジット表記要否
- 再配布可否
- 改変可否
- 有料版/無料版の差分（機能・利用条件）

## 補足
- 本一覧は「初期モックを速く作る」目的のための候補集。
- 本番利用時は、必ず各配布元の最新ライセンス条件を確認すること。
